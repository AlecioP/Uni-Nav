KEY VALUE
Nome : richiediDomanda()
Riferimenti : Gestione Prenotazioni
Pre-condizioni :
  • Lo studente deve aver precedentemente superato il limite massimo di biglietti non obliterati
  • Il Sistema deve conoscere le credenziali dello Studente che vuole effettuare domanda
  • Il Sistema deve conoscere almeno un Amministratore per potergli inoltrare la domanda
  • Il Sistema deve conoscere la data e l'ora correnti
Post-condizioni :
  • Viene creata un'istanza di Domanda
  • Viene creata un'associazione tra la nuova Domanda e lo Studente richiedente
  • Viene creata un'associazione tra la nuova Domanda e l'Amministratore alla quale é stata inviata quest'ultima
